테두리관련 스타일

        선택자 {
            * 테두리 스타일
            border[-위치]-style:none|dotted|dashed|double|groove|ridge|inset|outs

            * 테두리 두께
            border[-위치]-width:테두리두께;

            * 테두리 색상
            border[-위치]-color:테두리색상;

            * 테두리 두께, 스타일, 색상을 한꺼번에 적용
            border : 두께, 스타일, [색상]; 색상은 안넣으면 기본값 검정

            * 테두리 모서리 둥그렇게
            border-radius : 반지름;

            * 박스에 그림자 효과
            box-shadow : 가로거리(x) 세로거리(y) 흐림정도 번짐정도 색상;
        }
