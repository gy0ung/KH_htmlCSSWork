* jQuery 구문 작성하기

jQuery구문 dom요소를 다루는 구문을 기본적으로 작성하기 때문에
문서상에 요소들이 다 만들어지고 나서 실행되어야 함

> 순수 자바스크립트 에서는 

        window.onload = function() {
            실행내용;
        }

        javascript에서의 window.onload구문은
        해당문서의 요소들이 다 로드되고 연동되는
        외부문서들도 모두 가지고 온 후 실행 됨.
        window.onload는 단 한번만 실행.
> jQuery에서의 해당 내용은 3가지 구문으로 제공 
        1) jQuery(document).readt(function() {
            실행내용;
        })
        2) $(document).ready(function() {
            실행내용;
        })
        3) $(function() {
            실행내용;
        })
해당문서의 요소들만 다 로드되면 곧바로 실행됨. 여러번 기술가능(제한 없음) / 여러번 호출해도 됨. 순서대로 실행
