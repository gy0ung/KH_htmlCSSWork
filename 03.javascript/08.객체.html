객체

* 객체 선언 및 사용

객체는 중괄호{ }를 이용하여 생성하고,
중괄호 안에 이 객체에 필요한 속성들을 정의함
속성명:속성값 의 형태로 정의함(key:value)
속성값으로는 모든 자료형의 값을 담을 수 있음

        변수 = {
            속성명:속성값,
            속성명:속성값,
            ...
        }

* 객체 생성시 특이케이스(속성명 제시시 반드시 문자열의 형태로 해야되는 경우)

* 객체에서의 반복문

객체가 가지고 있는 모든 속성들에 순차적으로 접근하고자 한다면 반복문 사용 가능 단, 단순한 for loop로는 불가능 for in으로 사용가능
