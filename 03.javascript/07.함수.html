함수

* 선언적 함수

    function 함수명 ([매개변수,...]) {
        해당함수 호출시 실행할 코드;
        [return 결과값;]
    }

* 익명 함수(이름이 없는 함수)

    변수|속성 = function([매겨변수,....]){
        해당함수 호출시 실행할 코드;
        [return 결과값;]
    }
    
    특정 변수나 속성에 대입되는 함수 제시시 주로 사용
    (주로 이벤트 핸들러 작성시 사용)

* 화살표 함수(Arrow function)

    - 본문에 리턴만 있을 때
      변수|속성 = ([매개변수,...]) => 리턴값 (return쓰지않고 바로 리턴값을 써주면됨)
    
    - 본문이 여러줄 일 때
      변수|속성 = ([매개변수,...]) => {
        해당함수 호출시 실행할 코드;
        [return 결과값;]
    }
    
    * 익명함수를 간단히 작성하는 방법(버전 ES6(ES2015)에 추가)

* 즉시실행 함수(IIFE(immediately Invoked Function Expression))

    - 정의하면서 즉시 실행되는 함수
    ([변수] = function([매개변수,...]) {
        해당 함수 호출시 실행할 코드;
    })(인자,...);
    - 단 각각 한번만 호출됨

* 매개변수 지정방법

    - 매개변수가 없을 경우 :        () => return값; (매개변수의 소괄호 반드시 있어야 함) / return이라고 안쓰고 바로 리턴값
    - 매개변수 1개 일 경우 :        x => return값; (매개변수의 소괄호 생략가능)
    - 매개변수가 여러개일 경우 :    (x, y) => return값; (매개변수의 소괄호 반드시 있어야 함)

* 함수의 body 지정 방법

    - single line block
        x => { return 값 }
        x => return 값  (중괄호 생략가능)

    - multi line block
        x => {
            실행 구문들;
            return값; // 반드시 return키워드를 넣어줘야함
        }
