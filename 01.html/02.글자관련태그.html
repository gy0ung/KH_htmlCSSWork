문단을 나누는 태그 : p, pre

문단 영역을 나누는 태그로는 p 태그와 pre 태그가 있다. p 태그는 줄바꿈을 하고자 한다면 별도의 태그를 작성해야한다.
그리고 공백은 한개의 공백만을 표시하기 때문에 별도의      기호를 기술해야한다.

XML / XHTML에서 모든 태그는 끝 슬래시 … />로 자체를 닫는다.
XML을 테스트할때 서버가 올바른 인터넷 미디어 유형을 XHTML (예 : application/xhtml+xml)으로 보내고 있는지 확인해야 한다.
그렇지 않으면 브라우저에서 파일의 DOCTYPE에 관계없이 HTML로 처리한다. 

pre 태그는 시작태그부터 종료태그까지 입력된 내용 그대로 출력해줌

그 밖에 글자를 다루는 태그들

일반 글꼴 (태그로 감싸지 않는 텍스트)

b : (bold) 글자를 굵게 표시하는 태그

> strong : 글자를 굵게 표시하는 태그 + 스크린리더에서 강조해서 읽어줌

i : 글자를 기울여서 표시하는 태그

em : 글자를 기울여서 표시하는 태그 + 스크린리더에서 표시

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

s: 글자에 취소선 넣어주는 태그

small : 글자를 작게 표현해주는 태그

기본글자에 sub : 아래첨자 를 나타내는 태그
sup : 윗첨자 를 나타내는 태그

HTML5란 월드 와이드 웹 (world wide web)의 핵심 마크업 언어이다.
